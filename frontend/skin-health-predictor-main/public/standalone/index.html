📄 About This Project

Project Overview

This is a final-year machine learning project for Early-Stage Skin Disorder Detection. The system uses a trained classification model to predict one of 7 skin disorder categories based on clinical input parameters.

Model Details

  • Accuracy: ~91% on test data
  • Input Features: 10 clinical parameters (rated 0–4) + patient age
  • Output: Predicted disorder class (0–6) with probability score
  • Classes: Psoriasis, Seborrheic Dermatitis, Lichen Planus, Pityriasis Rosea, Chronic Dermatitis, Pityriasis Rubra Pilaris, Normal

Disorder Classes

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python FastAPI (for production deployment)
  • ML Model: Scikit-learn classifier (serialized with joblib)
  • Database: SQLite for prediction logging
⚠️ Disclaimer: This is an academic demonstration project. The predictions should not be used for actual medical diagnosis. Always consult a qualified dermatologist.
